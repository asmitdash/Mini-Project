1. Question: What is the mean (average) of the following data set: 5, 10, 15, 20, 25?
   - A) 5
   - B) 10
   - C) 15
   - D) 15
   - Correct Answer: C) 15

2. Question: What is the mode of the following data set: 4, 7, 7, 9, 12, 12, 15?
   - A) 7
   - B) 12
   - C) 15
   - D) There is no mode.
   - Correct Answer: B) 12

3. Question: If the standard deviation of a data set is 0, what can be said about the data set?
   - A) The data set is perfectly normally distributed.
   - B) The data set contains only one value.
   - C) The data set has a very large range.
   - D) The data set is not valid; standard deviation cannot be 0.
   - Correct Answer: B) The data set contains only one value.

4. Question: What is the median of the following data set: 8, 10, 12, 14, 16, 18, 20?
   - A) 10
   - B) 12
   - C) 14
   - D) 16
   - Correct Answer: C) 14

5. Question: In a normal distribution, what percentage of data points fall within one standard deviation from the mean?
   - A) 25%
   - B) 50%
   - C) 68%
   - D) 95%
   - Correct Answer: C) 68%
